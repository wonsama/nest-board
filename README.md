# nest-board

nest js 게시판 만들기

## 의존성 추가 설치

```sh
npm install --save @nestjs/typeorm typeorm mysql2 uuid
```

## backend

nestjs

### 설치

```sh
$ npm i -g @nestjs/cli
$ nest new backend
```

## 모듈 만들기

```sh
$ nest g module {모듈명}
$ nest g controller {모듈명}
$ nest g service {모듈명}

# 테스트 코드 (spec.ts) 없이 생성하고자 한다면
$ nest g controller {모듈명} --no-spec
```
